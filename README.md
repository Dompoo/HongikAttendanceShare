# 홍익대학교 출결번호 공유 시스템

## 기능
### 출결번호 검색
* 다른 사용자가 등록한 출결번호 조회
* 학수번호로 검색
* 포인트 1 차감
* 등록된 정보가 없다면 포인트 차감은 없음
* 같은 정보로 재검색 버튼

### 출결번호 등록
* 출결번호, 학수번호를 등록
* 임시 레포지토리에 저장해놨다가 다른 사용자가 똑같은 정보를 입력하면 영구 레포지토리로 이동
* 영구 레포지토리로 이동 시 포인트 2 추가
